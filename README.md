# vim
## vim command
### 移動系
- 先頭へ  : `gg`   
- 終わりに : `G`  
 
- 画面単位  
`Ctrl + b` / `Ctrl + f`   

- 画面上  
`H`: Home / `M`: Middle / `L`: Last  

- 単語単位  
`w` / `b`  

- 行頭 / 行末  
`^` / `$`  

- 文字までジャンプ  
`f(文字)` `;`で次の文字に  

- 対応する{}にジャンプ: `%`  

- *行目にジャンプ: `**G`  

### 選択系(visual mode) :`v`  
- 行単位  : `V`  
- 矩形単位: `Ctrl + v  `

### カット、コピー、貼付け
- カット : `x` / `dd` / `*dd`  
- コピー : `yy`  
- 貼付け: `p`
